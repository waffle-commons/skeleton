🦁 Waffle Skeleton
==================

Welcome to your new **Waffle** application.This skeleton provides the minimal structure to build robust, secure, and fast PHP applications using the Waffle Ecosystem.

🚀 Getting Started
------------------

### Prerequisites

*   Docker & Docker Compose

*   (Optional) PHP 8.3+ and Composer locally


### Installation

### 🐳 Run with Docker (FrankenPHP)

Waffle runs natively on FrankenPHP (Caddy). No external web server configuration is required.

📁 Structure
------------

*   app/: Your application code (Controllers, Services).

*   config/: Configuration files (app.yaml).

*   public/: Web entry point.

*   tests/: Your test suite.


🛡️ Security
------------

This skeleton is configured securely by default.See SECURITY.md for our disclosure policy.
